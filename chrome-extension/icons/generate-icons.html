Usage Instructions:

  1. Click the "Generate Icons" button
  2. Right-click each icon and select "Save Image As"
  3. Save with the corresponding filename: icon-16.png, icon-32.png, icon-48.png, icon-128.png
  4. Place the icon files in the firefox-extension/icons/ directory
